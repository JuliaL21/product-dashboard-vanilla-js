# Product Dashboard (Vanilla JS)

A simple product dashboard that fetches and displays store products from an external API using HTML, CSS, and vanilla JavaScript.

## Features
- Fetches product data from the [Course API Store Products](https://www.course-api.com/javascript-store-products)
- Promise-based fetch using `fetch()`, `.then()`, and `.catch()`
- Async/await fetch with `try`/`catch`
- Renders the first 5 products with name, image, and price
- Reusable `handleError()` function for error handling
- Hover effect on product cards (scale + shadow via CSS `:hover`)
